What is this page?

This is a website I made to collate my projects and involvements using CSS, JS, and HTML.
I've stored information on my python Habit Tracking App StreakIt, my FLORIS flowermaking program, my website on foraging, my musical interests and band, Shuttle.

I plan to leave this code open source as a possible resource for others to clone and display their own achievements, or collate the achievements of others.

Programming

Through my digitech classes at school, I've worked on a few projects, including a Python flower-making program, a website on foraging, and a habit tracking app called Streak It.
